    # Learn SVN Here!

    Simple Subversion

    # SVN 简介

    Subversion(SVN)是一个开源的版本控制系统，管理者随时间改变的数据。数据放置在一个中央档案库(repository)中。这个档案很像一个普通的文件服务器，但是他会记住每一次文件的改动。

    • Repository（源代码库）: 源代码统一存放的地方
    • Checkout（提取）: 从仓库获取源代码
    • Commit（提交）:上传已经修改的的代码
    • Update（更新）:同步仓库代码，使得本地代码保持最新

    # SVN 安装

    执行svn --version查看SVN版本，确认是否已经存在SVN，不存在则进行下载

    • Windows（下载）: https://sourceforge.net/projects/win32svn/
    • CentOS（自带）: yum install subversion
    • Ubuntu（自带）: apt-get install subversion
    • MacOS（自带）: brew install subversion

    # SVN 检出

    # 无密码
svn checkout http://svn.server.com/svn/project_repo--username=user01

# 有密码
svn checkout --username=xxx --password=xxx svn://domain@xx.xx.xx.1:port/project

# 指定版本
svn checkout -r r21587 --username=xxx --password=xxx svn://xxx@11.111.111.1:6000/project

    # SVN 提交

    # 在版本库中增加一个readme文件
cat readme
=> Hello

# 查看本地文件状态
svn status
=> ? readme

# 将文件加入版本控制
svn add readme
=> A readme

# 查看文件状态
=> A readme

# 提交更改
svn commit -m "init svn readme"

    # 回退

    未提交

    # 查看文件状态
svn status
=> M readme

# 撤销文件更改
svn revert readme
Reverted 'readme'

# 确认文件状态
svn status

# 撤销文件夹更改
svn revert -R trunk

    已提交

    # 22是当前版本，21是回退版本，readme是回退的文件路径
svn merge -r 22:21 readme

    # SVN 信息

    通过svn命令根据时间或者版本号去除过去的版本

    • svn log： 展示版本、作者、日期、路径
    • svn diff： 像是修改的行级信息
    • svn cat： 取得某版本的某文件显示
    • svn list： 显示一个目录或者某一版本存在的文件
